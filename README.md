## How to Build and Deploy

Best Practice is to use [Dockerfile](dockerfile)

```powershell
# remove all running docker containers
docker container rm $(docker container ls -aq) -f

# build and run docker container
# http://localhost:1111
npm run docker-run
```


## Tester & QE
